# Nani's Knees — Landing Page

The official marketing site for **Nani's Knees Strength Blueprint**, a 12-week strength training program for women.

Live at: [naniknees.com](https://naniknees.com)

---

## About This Repo

Single-page marketing site built in vanilla HTML and CSS. No framework, no build step, no dependencies. Designed to load instantly and render beautifully on every device.

- **Landing page** — `index.html`
- **404 page** — `404.html`
- **Deployment config** — `vercel.json`

## Design System

- **Type:** Fraunces (display serif) + Inter (body sans-serif), loaded from Google Fonts
- **Palette:** Oxblood, bordeaux, terra, flesh, silk, and gold over a dark ink base
- **Aesthetic:** Editorial magazine — warm, confident, feminine

## Deployment

This site auto-deploys to [Vercel](https://vercel.com) on every push to `main`.

Local preview: open `index.html` directly in a browser. No build step needed.

## Updates

To change the site, edit `index.html` and commit. Vercel will redeploy within ~30 seconds.

Common edits:
- **Pricing** — search for `price-amount` in the file
- **Copy** — the manifesto, hero, and FAQ sections are clearly labeled with HTML comments
- **CTAs** — replace `href="#"` on the three `.price-cta` buttons with your Gumroad / Stripe URLs
